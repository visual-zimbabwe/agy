# Quick Capture Power User Guide

## Fastest Workflow

1. Open quick capture quickly  
Use `Q` (or `Ctrl/Cmd + J`) so you can stay on keyboard.

2. Think in lines, not notes  
Write one idea per line. Each line becomes one note when captured.

3. Tag inline while typing  
Use `#tags` directly in each line:
- `Draft onboarding checklist #ops #onboarding`

Tags are parsed automatically into note tags.

4. Batch capture in one action  
Press `Ctrl/Cmd + Enter` to create all notes at once.

5. Use paste for high-throughput import  
Paste bullet lists from docs, Slack, email, or meeting notes into quick capture, then capture.

List markers are parsed and each line becomes a note.

6. Use voice for raw capture, keyboard for refinement  
Tap `Voice to Notes`, dictate short lines, and continue after short pauses if needed (silence timeouts auto-retry). Stop when done, then quickly clean text/tags before capture.

7. Organize immediately with multi-select  
After capture, box-select new notes and apply:
- bulk color
- bulk tags
- align/distribute

## Practical Line Patterns
- `Idea text #theme #priority`
- `Action text #owner-jane #next-week`
- `Risk text #blocker #needs-review`

This flow maximizes capture speed while preserving structure for search and organization.
