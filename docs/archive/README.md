# Archive

## Purpose

This folder contains preserved historical documentation that is no longer part of the active canonical documentation surface.

## Status Rules

Archived docs:

- may contain discovery notes, implementation plans, or superseded context
- are retained for project history and decision traceability
- are not the current source of truth for product, architecture, API, or contributor workflow

## Current Canonical Docs

Use the active canonical docs under:

- `docs/product/`
- `docs/architecture/`
- `docs/features/`
- `docs/api/`
- `docs/decisions/`
- `docs/runbooks/`
- `docs/contributing/`
- `docs/releases/`

## Legacy Plans

`docs/archive/legacy-plans/` contains older planning, discovery, and implementation files that were previously mixed into the top-level `docs/` directory.
